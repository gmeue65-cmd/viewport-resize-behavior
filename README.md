# 📊 Analizador Interactivo de Raíces Cúbicas en el Plano Complejo

Este proyecto implementa un entorno de análisis numérico robusto para el cálculo y la visualización interactiva de raíces reales y complejas en polinomios cúbicos. Combina la aproximación global por autovalores de **NumPy** con la convergencia local de alta velocidad del método **Newton-Raphson (SciPy)**.

El sistema cuenta con un algoritmo de **zoom adaptativo e inteligente** que unifica la posición de las raíces y el origen cartesiano $(0,0)$, garantizando un encuadre geométrico perfecto (`aspect ratio 1:1`) sin importar la escala de los coeficientes.

## 🚀 Características
* **Cálculo Híbrido:** Resolución global simultánea de componentes reales e imaginarias.
* **Lienzo Adaptativo:** Escalamiento automático del lienzo basado en el 20% de la dispersión máxima global (raíces + origen).
* **Precisión Visual:** Cuadrícula interactiva dinámica de alta resolución mediante `AutoMinorLocator`.
